1.Дать определение:

	Наследование - возможность создания класса который будет содержать в себе поля и методы другого класса (так называемого родительского).
	Абстрактный класс - Класс у которого хотя бы один виртуальный метод приравнен к нулю (Тоесть отсутствует в таблице виртуальных методов).
 Обьекты такого класса не могут быть созданы.
	Таблица виртуальных методов - содержит список методов для виртуальных функций класса. Именно в ней находится соответствие метода и его
"содержимого" при использовании динамического полиморфизма.
	Поле - это минимальная единица содержимого класса. Поле может быть переменной или методом. Поля могут быть public - к таким полям класса возможен свободный доступ
из всей программы, private - к таким полям имеют доступ только методы класса, protected - выполняет ту же функцию, что и private ,
за исключением того, что к этим полям могут иметь доступ классы отнаследованные от данного класса.
	Список инициализации - присвоение значений переменным класса в конструкторе вне его тела в виде списка (через запятую).
	Экземпляр класса - обьект (имеющий имя) обладающий всеми полями и методами данного класса.
	Полиморфизм - выполнение различных действий для разных обьектов при использовании одного оператора или метода. 
	
2. 	Внутри cpp файлов необходимо подключить необходимые для его работы заголовочные файлы.
	В h файлы желательно не подключать ничего, так как это увеличивает зависимости программы.

	class String
	{
		private:
			unsigned int length;
			char* characters;
		public:
			String();
			String(String&);
			const char*get() const;
	}

	String::String():length(0),characters(nullptr)
	{
	}

	String::String(String& another_string):length(0),characters(nullptr)
	{
		characters=new char sizeof[];
		strcopy(characters,another_string.characters);
